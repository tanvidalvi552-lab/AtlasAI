Portfolio Description

AtlasAI is a premium AI-inspired travel planning platform that helps users discover destinations, visualize itineraries, estimate budgets, and explore curated experiences through a modern interactive interface. Built with React and advanced animations, the project focuses on delivering a seamless user experience while showcasing responsive design, reusable components, and visually engaging storytelling.
